# GeoPreservar - Consultoria Ambiental e Fundiária

Landing page profissional desenvolvida para a **GeoPreservar**, empresa especializada em geotecnologias, regularização de imóveis rurais e gestão ambiental estratégica. O projeto foi desenhado para converter visitantes em clientes, destacando a expertise técnica e a autoridade do especialista no mercado paraense.

## Tecnologias Utilizadas

Para garantir performance, SEO e uma interface moderna, foram utilizadas as seguintes tecnologias:

* **HTML5 Semântico:** Estruturação focada em acessibilidade e indexação.
* **Tailwind CSS:** Framework utilitário para estilização rápida, responsiva e otimizada.
* **JavaScript (Vanilla):** Lógica de interação para o menu mobile e comportamentos de scroll.
* **AOS (Animate On Scroll):** Biblioteca para animações de entrada que tornam a navegação mais fluida e dinâmica.
* **FontAwesome:** Ícones vetoriais para representação visual dos serviços.
* **Google Fonts:** Utilização da família *Inter* para máxima legibilidade.

---

## Funcionalidades e Diferenciais

* **Design Responsivo:** Adaptado para smartphones, tablets e desktops (Mobile-First).
* **Navegação Glassmorphism:** Menu fixo com efeito de desfoque (backdrop-filter) para uma estética moderna.
* **CTA Estratégico:** Botões de conversão direta para WhatsApp integrados em todas as seções.
* **Seções de Especialidade:** * Georreferenciamento de Imóveis Rurais (INCRA).
* Cadastro Ambiental Rural (CAR) e Retificações.
* Licenciamento Ambiental (SEMAS/IBAMA).
* Regularização Fundiária (ITERPA).
* Crédito Rural (PRONAF, FNO, ABC).


* **SEO Local:** Focado no mercado do Pará, destacando a experiência do especialista junto aos órgãos estaduais.
